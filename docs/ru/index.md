## XTLS? Xray? V2Ray?

> **XTLS — это блестящая идея для TLS, которую мы изучаем, в то время как Xray — это лучшая практика, которую мы поддерживаем.**

- Xray-core изначально вырос из v2ray-core, но уже давно развивается самостоятельно, поэтому его больше не стоит считать полностью совместимой заменой v2ray-core.
  - Только один исполняемый файл, включающий функциональность ctl, команда run используется по умолчанию.
  - Часть структуры конфигурации и привычных способов использования похожа на v2ray-core, но переменные среды, префиксы API и многие возможности уже отличаются.
  - Открытый raw протокол ReadV на всех платформах.
  - Полная поддержка VLESS, XTLS Vision и REALITY.
  - Доступно несколько режимов управления потоком XTLS, а в подходящих сценариях вместе со Splice это дает очень высокую производительность.
